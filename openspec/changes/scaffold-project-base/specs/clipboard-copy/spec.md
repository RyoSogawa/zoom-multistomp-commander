# Capability: clipboard-copy

生成されたSysExメッセージをクリップボードにコピーする機能。

## ADDED Requirements

### Requirement: ワンタッチコピー

ユーザーは生成されたメッセージをワンタッチでクリップボードにコピーできなければならない（SHALL）。

#### Scenario: コピーボタンをタップ

- Given: SysExメッセージが生成されて表示されている
- When: ユーザーがコピーボタンをタップする
- Then: メッセージがクリップボードにコピーされる
- And: コピー成功のフィードバックが表示される

### Requirement: コピー成功フィードバック

コピー成功時にユーザーに視覚的なフィードバックを提供しなければならない（SHALL）。

#### Scenario: コピー成功時のフィードバック表示

- Given: ユーザーがコピーボタンをタップした
- When: クリップボードへのコピーが成功する
- Then: ボタンまたはUIにコピー成功の表示が一時的に現れる
- And: 数秒後に元の状態に戻る

### Requirement: コピー失敗時のエラーハンドリング

クリップボードAPIが利用できない場合、適切なエラーメッセージを表示しなければならない（SHALL）。

#### Scenario: クリップボードAPIが利用不可

- Given: ブラウザがクリップボードAPIをサポートしていない
- When: ユーザーがコピーボタンをタップする
- Then: エラーメッセージが表示される
